## 🎯 How to Use

1. **Browse** the directories to find wallpapers that suit your taste
2. **Download** any image by clicking on it and selecting "Download"
3. **Set as Wallpaper** on your device:
   - **Windows**: Right-click image → "Set as desktop background"
   - **macOS**: System Preferences → Desktop & Screen Saver → Select image
   - **Linux**: Right-click on desktop → Set as wallpaper (varies by DE)
